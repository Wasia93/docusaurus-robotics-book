# 欢迎来到物理AI与人形机器人

这是一门综合课程，从ROS 2基础知识到构建自主人形机器人。

## 您将学到什么

在本课程中，您将学习：

- **ROS 2基础**：理解机器人操作系统2
- **物理仿真**：在Gazebo和Unity中创建逼真的环境
- **NVIDIA Isaac**：使用Isaac Sim和Isaac ROS进行AI驱动的机器人技术
- **语音控制**：将GPT/Claude与Whisper集成用于自然语言机器人控制
- **双足运动**：实现行走算法和平衡控制
- **人机交互**：设计自然交互系统

## 课程结构

课程分为四个主要模块：

1. **基础**：物理AI简介和人形机器人概况
2. **ROS 2和中间件**：掌握机器人操作系统2
3. **仿真和NVIDIA Isaac**：为真实世界创建数字孪生
4. **视觉-语言-动作**：构建完全自主的人形机器人

## 开始学习

使用左侧边栏中的导航浏览课程内容。我们建议您从头开始并按顺序进行。

每个模块包括：
- 详细的课程和教程
- 实践练习
- 代码示例
- 真实世界项目

让我们开始您的物理AI和人形机器人之旅！
