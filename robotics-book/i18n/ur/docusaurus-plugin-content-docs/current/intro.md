# فزیکل AI اور ہیومنائڈ روبوٹکس میں خوش آمدید

یہ ایک جامع کورس ہے جو آپ کو ROS 2 کی بنیادی باتوں سے لے کر خود مختار ہیومنائڈ روبوٹس بنانے تک سکھاتا ہے۔

## آپ کیا سیکھیں گے

اس کورس میں آپ یہ سیکھیں گے:

- **ROS 2 بنیادیں**: Robot Operating System 2 کو سمجھیں
- **فزکس سمولیشن**: Gazebo اور Unity میں رئیلسٹک ماحول بنائیں
- **NVIDIA Isaac**: AI سے چلنے والے روبوٹکس کے لیے Isaac Sim اور Isaac ROS استعمال کریں
- **وائس کنٹرول**: قدرتی زبان کے روبوٹ کنٹرول کے لیے GPT/Claude کو Whisper کے ساتھ ضم کریں
- **بائپیڈل لوکوموشن**: چلنے کے الگورتھم اور توازن کنٹرول لاگو کریں
- **انسان-روبوٹ تعامل**: قدرتی تعامل کے نظام ڈیزائن کریں

## کورس کی ساخت

یہ کورس چار اہم ماڈیولز میں تقسیم ہے:

1. **بنیادیں**: فزیکل AI کا تعارف اور ہیومنائڈ روبوٹکس کا منظرنامہ
2. **ROS 2 اور مڈل ویئر**: Robot Operating System 2 میں مہارت حاصل کریں
3. **سمولیشن اور NVIDIA Isaac**: حقیقی دنیا کے لیے ڈیجیٹل ٹونز بنائیں
4. **Vision-Language-Action**: مکمل خود مختار ہیومنائڈ روبوٹس بنائیں

## شروع کرنا

بائیں طرف سائڈبار میں نیویگیشن استعمال کریں تاکہ کورس کے مواد کو دریافت کریں۔ ہم تجویز کرتے ہیں کہ آپ شروع سے شروع کریں اور ترتیب سے آگے بڑھیں۔

ہر ماڈیول شامل ہے:
- تفصیلی سبق اور ٹیوٹوریلز
- عملی مشقیں
- کوڈ کی مثالیں
- حقیقی دنیا کے پروجیکٹس

آئیے اپنا سفر فزیکل AI اور ہیومنائڈ روبوٹکس میں شروع کریں!
