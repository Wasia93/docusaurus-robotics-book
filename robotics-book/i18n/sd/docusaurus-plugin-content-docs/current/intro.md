# فزيڪل AI ۽ هيومنائيڊ روبوٽڪس ۾ ڀلي ڪري آيا

هي هڪ جامع ڪورس آهي جيڪو توهان کي ROS 2 جي بنيادي ڳالهين کان وٺي خودمختيار هيومنائيڊ روبوٽس ٺاهڻ تائين سيکاري ٿو.

## توهان ڇا سکندا

هن ڪورس ۾ توهان سکندا:

- **ROS 2 بنياد**: Robot Operating System 2 کي سمجھو
- **فزڪس سموليشن**: Gazebo ۽ Unity ۾ حقيقي ماحول ٺاهيو
- **NVIDIA Isaac**: AI سان هلندڙ روبوٽڪس لاءِ Isaac Sim ۽ Isaac ROS استعمال ڪريو
- **آواز ڪنٽرول**: قدرتي ٻولي روبوٽ ڪنٽرول لاءِ GPT/Claude کي Whisper سان گڏ ڪريو
- **ٻه پيرن واري حرڪت**: هلڻ جا الگورٿم ۽ توازن ڪنٽرول لاڳو ڪريو
- **انسان-روبوٽ رابطو**: قدرتي رابطي جا نظام ڊزائين ڪريو

## ڪورس جي جوڙجڪ

اهو ڪورس چار اهم ماڊلز ۾ ورهايل آهي:

1. **بنياد**: فزيڪل AI جو تعارف ۽ هيومنائيڊ روبوٽڪس جو منظرنامو
2. **ROS 2 ۽ مڊل ويئر**: Robot Operating System 2 ۾ مهارت حاصل ڪريو
3. **سموليشن ۽ NVIDIA Isaac**: حقيقي دنيا لاءِ ڊجيٽل ٽوئنز ٺاهيو
4. **Vision-Language-Action**: مڪمل خودمختيار هيومنائيڊ روبوٽس ٺاهيو

## شروع ڪريو

کاٻي پاسي سائڊبار ۾ نيويگيشن استعمال ڪريو ڪورس جي مواد کي دريافت ڪرڻ لاءِ. اسان سفارش ڪريون ٿا ته توهان شروع کان شروع ڪريو ۽ ترتيب سان اڳتي وڌو.

هر ماڊل شامل آهي:
- تفصيلي سبق ۽ ٽيوٽوريل
- عملي مشقون
- ڪوڊ جا مثال
- حقيقي دنيا جا منصوبا

اچو ته شروع ڪريون پنهنجو سفر فزيڪل AI ۽ هيومنائيڊ روبوٽڪس ۾!
